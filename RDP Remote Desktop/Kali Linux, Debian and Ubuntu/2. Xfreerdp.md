## ğŸ› ï¸ Basic Command Structure

```bash
xfreerdp /u:<username> /p:<password> /v:<server_address>
```

**Example:**

```bash
xfreerdp /u:nikhil /p:7489 /v:192.168.95.144
```

* `/u:` = Username
* `/p:` = Password
* `/v:` = IP or hostname
